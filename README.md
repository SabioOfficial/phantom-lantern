Forged with one **Phantom Membrane**, and a **Soul Lantern**, you can create the **Phantom Lantern**. It is a _very_ helpful support weapon that does **passive damage**, gives you **insane mobility**, **_and more!_**

![hr](https://cdn.modrinth.com/data/cached_images/f9ef6b8eb5bc66fcf87cd276409f6705a1fbc79b.png)

![Features](https://cdn.modrinth.com/data/cached_images/4517bf16c9f8721c3b11070eeab1e32dd133beb1.png)

- 🌀 **Soul System** (Mana-like System)
- ✦ **Essence System** (You use this to gain new abilities)
- 👻 **Haunting** (Passive Damage)
- 👞 **Phantom Step** (Dash **_through_** 1-2 block walls!)
- 🚧 **Spectral Barrier** (Applies **Slowness I** to all entities for a period of time)
